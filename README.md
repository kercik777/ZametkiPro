# 📝 ЗаметкиPro

> Премиум приложение для заметок с чёрно-золотым дизайном

Современное Android-приложение на Java (без Kotlin, без ViewBinding), целиком на русском языке, с локальным хранением данных (SQLite + SharedPreferences). Готово к публикации в Google Play.

---

## ✨ Особенности

- 🎨 **Премиум дизайн** — чёрно-золотая палитра (Black & Gold Luxury)
- 📝 **Текстовые заметки и чек-листы** — с конвертацией одного типа в другой
- 🏷 **Категории и цветовые метки** — 8 категорий + 9 цветов заметок
- 📌 **Закрепление и избранное** — для важных заметок
- 🔍 **Мощный поиск** — по заголовку, контенту и пунктам чек-листов
- 🗂 **Архив и Корзина** — с возможностью восстановления
- 📊 **Подробная статистика** — с анимированными графиками распределения
- 🌓 **Тёмная / Светлая / Системная тема**
- 🎬 **Плавные анимации** — slide, fade, layout animations
- 📤 **Экспорт заметок** — в текстовый файл с возможностью поделиться
- 📱 **Адаптивная иконка** — Material You ready
- 🔒 **100% локальное хранение** — никаких сетевых разрешений
- 🇷🇺 **Полностью на русском языке** (с готовой локализацией структуры под en)

## 🛠 Технологии

- **Язык:** Java 17
- **min SDK:** 26 (Android 8.0)
- **target SDK:** 34 (Android 14)
- **БД:** SQLite (через `SQLiteOpenHelper`)
- **UI:** AndroidX + Material Components
- **Без сторонних библиотек** — только базовый AndroidX и Material
- **Без ViewBinding** — классический `findViewById`
- **Системные шрифты** — `sans-serif` (без TTF)

## 🚀 Как открыть проект в Android Studio

### Способ 1 — Импорт существующего проекта

1. Скачайте папку `ZametkiPro/` целиком к себе на компьютер.
2. Откройте **Android Studio** (рекомендуется Hedgehog 2023.1.1 или новее).
3. На стартовом экране выберите **«Open»** (или `File → Open`).
4. Выберите папку `ZametkiPro` и нажмите **OK**.
5. Android Studio спросит про **Gradle Wrapper** — нажмите **OK / Trust Project**.
6. Дождитесь завершения **Gradle Sync** (~1–3 минуты, скачаются зависимости).
7. Нажмите ▶ **Run** или `Shift+F10` — приложение запустится.

### Способ 2 — Создать пустой проект и заменить файлы

1. В Android Studio: `New Project → Empty Views Activity`
2. Имя: `ЗаметкиPro`, package: `com.zametki.pro`, language: **Java**, min SDK: **26**.
3. После создания — закройте студию, замените содержимое папки на содержимое из этого проекта.
4. Откройте проект заново → Sync Gradle.

## ⚙️ Если Gradle Sync падает

| Проблема | Решение |
|---|---|
| Не находит JDK 17 | `File → Settings → Build, Execution, Deployment → Build Tools → Gradle → Gradle JDK` → выбрать **JBR 17** |
| Не качается Gradle 8.2 | Проверь интернет / прокси. Можно вручную поменять версию в `gradle/wrapper/gradle-wrapper.properties` |
| Ошибка `compileSdk 34` | `Tools → SDK Manager` → установи **Android SDK Platform 34** |
| Кириллица в `settings.gradle` | Если IDE ругается — переименуй `rootProject.name` на латиницу: `'ZametkiPro'` |

## 🔧 Подготовка к публикации в Google Play

1. **Подпиши APK / AAB:**
   - `Build → Generate Signed Bundle / APK → Android App Bundle`
   - Создай `keystore` (сохрани пароль!).
2. **Поменяй `applicationId`** в `app/build.gradle` на свой уникальный (например, `com.yourname.zametkipro`).
3. **Иконки** уже готовы (адаптивные mipmap-anydpi-v26).
4. **Разрешения** — только `VIBRATE` (не требует runtime-запроса). Никаких чувствительных permissions, идеально для модерации.
5. **Backup rules** — настроены, поддерживают резервное копирование БД и настроек.
